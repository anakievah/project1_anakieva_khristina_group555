# project1_anakieva_khristina_group555
# Лабиринт сокровищ

# Консольная игра на Python, в которой игроку необходимо пройти лабиринт и найти сокровище.

# Проект выполнен в рамках учебного задания с использованием: 
# * Python
# * Poetry
# * Git / GitHub

# Описание игры

# Игрок управляет персонажем (`P`), который перемещается по лабиринту.
# Цель игры — **найти сокровище (`T`)**, избегая стен (`#`).

# Игра завершается: при нахождении сокровища, при выходе игрока по команде `q`

# Управление
| Клавиша | Действие      |
| ------- | ------------- |
| `w`     | Вверх         |
| `s`     | Вниз          |
| `a`     | Влево         |
| `d`     | Вправо        |
| `q`     | Выход из игры |

# Обозначения в лабиринте
| Символ | Значение      |
| ------ | ------------- |
| `P`    | Игрок         |
| `#`    | Стена         |
| ` `    | Пустая клетка |
| `T`    | Сокровище     |
